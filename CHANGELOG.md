# Changelog
All notable changes to VishwaGuru will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- 🤖 AI-powered action plans using Google Gemini (WhatsApp messages, email drafts, X posts)
- ⚡ FastAPI-powered backend with high-performance asynchronous API
- 🎨 Modern React + Vite frontend with responsive UI
- 📱 Telegram bot integration for direct issue reporting
- 🗄️ SQLite (dev) and PostgreSQL (prod) database support
- ☁️ Cloud-native design for Netlify, Render, and Neon deployment
- 📍 Spatial deduplication to detect and prevent duplicate nearby issues
- 🔍 Unified AI-powered detection (potholes, garbage, vandalism, etc.)
- 🏛️ Maharashtra MLA lookup by pincode with grievance filing
- 📚 Comprehensive documentation (Architecture, Deployment, Contributing)
- 🔒 Security policy and vulnerability reporting guidelines
- 📋 Issue and PR templates for streamlined contributions
- 📊 GitHub Actions CI/CD pipeline
- ✅ Pre-commit hooks for code quality
- 🧪 Unit and integration tests

### Changed
- Migrated from SQLite to PostgreSQL for production deployments
- Updated UI with modern Tailwind CSS design system
- Enhanced error handling and logging across all services

### Fixed
- Various deployment issues on Render and Netlify
- Mobile responsiveness improvements
- Database connection stability

## [1.0.0] - 2024-XX-XX

### Added
- Initial release of VishwaGuru
- Core civic issue reporting functionality
- AI-powered solution generation
- Basic web interface
